How It Works

Up and running in 3 steps

No complex setup. No cloud accounts. Just install, configure, and start chatting.

1

Install

One command installs everything. Works on Linux, macOS, and Windows. Docker support included.

2

Configure

Add your API keys (or use local models with Ollama). The setup wizard guides you through it.

3

Use

Chat via CLI, web UI, or any messaging platform. Your AI is ready to help with anything.
